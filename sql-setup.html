📖 How to Run

  1. Copy the SQL: Click the blue "Copy All SQL" button
  2. Open Supabase: Click "Open Supabase" (or go to app.supabase.com)
  3. SQL Editor: Select your GCR project → SQL Editor → New Query
  4. Paste & Execute: Paste the SQL, click Execute (blue button)
  5. Done! You should see "Query executed successfully"
