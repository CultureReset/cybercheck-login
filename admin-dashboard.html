Manage all 820 businesses from your Supabase database

📤 Bulk Upload Businesses via CSV

CSV Format Requirements:

Your CSV file should have these columns (in any order):

  • name * (required) - Business name
  • category * (required) - restaurants, coffee-sweets, nightlife, things-to-do, shopping, hotels, or other
  • address - Full address
  • phone - Phone number
  • website - Website URL
  • description - Short description
  • about - Detailed information
  • image - Main image URL
  • profile_pic - Profile picture URL
  • photos - Additional photo URLs (separate multiple with semicolons)
  • rating - Rating (0-5)
  • price_level - $, $$, $$$, or $$$$
  • hours - Operating hours
  • tags - Tags (separate multiple with semicolons)
  • has_happy_hour - true or false
  • featured - true or false
  • verified - true or false

💡 Tip: For multiple values (photos, tags), separate them with semicolons (;)
