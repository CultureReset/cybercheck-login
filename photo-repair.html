📷 Photo Repair

Some businesses have photo rows pointing at storage paths where the file was never actually uploaded (a legacy import problem, not something recent). Most of these already have real Google photo references on file — this tool fetches those and uploads them to the exact path each broken row expects, so the existing links heal. Nothing is guessed; unrecoverable rows (no Google photo available) are removed instead of left permanently broken.
