# Menu Systems Comparison

## 1. Menu Builder (admin dashboard)
**Location:** Cybercheck Login → Sidebar → "Menu Builder 🤖"
**Users:** Admin/staff only
**Purpose:** Comprehensive menu management for one business at a time

### Features:
- Business selector dropdown (search, manage one business)
- **Tabs:**
  - Menu Items: CRUD operations for categories, items, pricing
  - AI Scanner: Upload menu photos → AI extracts items
  - Events & Specials: Add events, specials, happy hour items
  - Theme Designer: Customize menu appearance
  - AI Design: Auto-generate menu layouts
  - Promotions: Setup coupon/loyalty triggers
- QR Code: Generate unique QR code for that business's table ordering
- Link to GCR: Connect menu to Google Business Profile

### Data Flow:
- Saves to: `/api/gcr/entities/:id` and related menu endpoints
- Fetches from: `/api/gcr/entity/:slug`
- Updates appear in: GCR profiles, QR menu links, anywhere that business's menu displays

---

## 2. QR Menus (admin dashboard)
**Location:** Cybercheck Login → Sidebar → "QR Menus"
**Users:** Admin/staff only
**Purpose:** Quick access to all restaurants' QR menu links + PIN/edit link management

### Features:
- List of ALL restaurants (searchable by name)
- For each restaurant:
  - **View Menu:** Opens the live QR menu (read-only)
  - **Copy URL:** Copy the QR menu link to share
  - **Print QR:** Generate printable QR code card
  - **PIN field:** Set a PIN for that business's editable menu
  - **Edit Link button:** Generate a shareable edit link that includes the PIN
    - Link format: `https://cybercheck-links.vercel.app/daily-menu.html?slug={slug}&pin={pin}`
  - **Test Link button:** Preview the edit link immediately

### Data Flow:
- Reads from: `/api/gcr/entities?limit=500`
- PIN stored in: Admin's local management (displayed on QR Menus page)
- Edit link generated by: `generateQREditLink()` function
- Business opens link at: cybercheck-links.vercel.app with PIN auto-filled

---

## 3. Menu Editors Hub (cybercheck-links)
**Location:** https://cybercheck-links.vercel.app/admin-menu-editors.html
**Users:** Admin/staff (easier access than login dashboard)
**Purpose:** Quick hub to access different menu editor types for any business

### Features:
- Searchable list of all restaurants
- For each restaurant, three editor types:
  1. **Choose (🎯)** → menu-editor-choose.html?slug={slug}
     - Full menu editor with category selection UI
  2. **Quick (📱)** → daily-menu.html?slug={slug}
     - Simple fast menu editor for daily updates
  3. **With Ads (📢)** → daily-menu-ads.html?slug={slug}
     - Menu editor with promotional content
  
- For each editor:
  - **Open link** button: Opens the editor
  - **Copy button (📋):** Copies URL to clipboard
  - **QR button (📱):** Shows QR code for the editor link

### Data Flow:
- Reads from: `/api/gcr/entities?limit=500`
- Links are relative/absolute URLs pointing to cybercheck-links pages
- No PIN system yet (raw URLs)

## Your Current Workflow:

1. **Admin goes to QR Menus page**
2. **Searches for business** (type in search box)
3. **Sets PIN for that business** (enter in PIN field)
4. **Clicks "Edit Link" button** → generates link with PIN embedded
5. **Clicks "Copy Link"** → copies to clipboard
6. **Sends link via SMS** to the business owner
7. **Business opens link** → PIN is auto-filled → edits menu → saves
8. **Updates appear everywhere** (GCR, QR menu, all connected systems)
